# Logos de Marcas

Coloca aquí los logos de las marcas en formato SVG o PNG.

## Estructura de archivos esperada:

```
public/images/brands/
├── daikin.svg (o .png)
├── mitsubishi.svg
├── lg.svg
├── samsung.svg
├── carrier.svg
├── york.svg
├── trane.svg
├── fujitsu.svg
├── panasonic.svg
├── midea.svg
├── gree.svg
└── hitachi.svg
```

## Opciones para obtener los logos:

### 1. **Sitios oficiales de las marcas**
   - Visita las secciones de "Prensa", "Medios" o "Recursos" de cada marca
   - Ejemplos:
     - Daikin: https://www.daikin.com/press/
     - LG: https://www.lg.com/global/business/press-resources
     - Samsung: https://news.samsung.com/

### 2. **Repositorios de logos gratuitos**
   - **Simple Icons**: https://simpleicons.org/ (tiene muchas marcas en SVG)
   - **Logo.dev**: https://logo.dev/ (API de logos)
   - **World Vector Logo**: https://worldvectorlogo.com/
   - **Seeklogo**: https://seeklogo.com/
   - **Logos Download**: https://logos-download.com/

### 3. **Bancos de imágenes**
   - Wikimedia Commons: https://commons.wikimedia.org/
   - Freepik: https://www.freepik.com/

### 4. **Contactar a las marcas**
   - Si eres distribuidor autorizado, contacta al departamento de marketing
   - Solicita los logos en formato SVG o PNG de alta resolución

## Recomendaciones:

- **Formato preferido**: SVG (escalable y ligero)
- **Tamaño**: Mínimo 200x200px para PNG
- **Fondo**: Preferiblemente transparente
- **Colores**: Logos en color original o versión monocromática según diseño

## Nota Legal:

Asegúrate de tener los derechos para usar estos logos. Si eres distribuidor autorizado, generalmente está permitido mostrar los logos de las marcas que representas.
